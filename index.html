HTML 학습을 위한 웹페이지 구성 실습. 웹사이트 구성의 기초적 단계 학습을 위한 Index Page

HTML은 웹페이지의 기초 뼈대를 구성하는데 사용되는 언어이다. 웹사이트는 웹 페이지가 여러 개 모인 집합이며 우리가 보는 모든 웹사이트는 HTML롤 구성된 여러 웹페이지의 모음이다.

CSS는 HTML로 만들어진 웹사이트에 살을 붙여 꾸며주는 역할을 하며, JavaScript는 만들어진 웹페이지안에서 사용자가 동적으로 상호작용할 수 있도록 만들어준다.

이러한 의미에서 HTML, CSS, JavaScript는 웹사이트를 구성하는데 있어 필수적인 3요소라고 할 수 있으며, 이렇게 사용자가 직접 보고 상호작용 할 수 있는 웹사이트를 만드는 영역을 Front-End라고 한다.
