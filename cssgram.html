2. class 지정과 선택자

class를 지정한다는 것은 곧 그룹을 지정한다는 의미를 나타낸다.
태그에 클래스를 지정하여 그룹을 형성하고 해당 그룹에 속성을 부여할 수 있다.
이를 통해 자신이 원하는 태그를 정교하게 targeting할 수 있다

class 값에는 여러 개가 사용될 수 있으며 띄어쓰기로 구분한다. 이때 가장 뒤에 있는,
즉 우선순위가 높은 클래스 값에 속성이 부여된다.
EX) class="saw active"라고 할때 saw와 active 각각에 속성을 부여할 수 있다.

a. id값 지정

class에 id값을 지정하면 순서 상관없이 id값이 부여된 클래스에 속성이 부여된다.

문법: id를 지정한 후 style 태그에서 #OO{font-size:10px;} 와 같은 형식으로 속성을 부여한다

b. 우선순위

inline 값> id값 > class 값 > 태그
조금 더 구체적인 값에 우선순위를 둔다.
